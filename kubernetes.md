## Kubernetes?
- Kubernetes adalah aplikasi open source untuk automating deployment, scaling dan manajemen aplikasi berbasis container.

- kube-controller-manager
- cloud-controller-manager
### Node Components
- kubelet
- kube-proxy
- Container runtime
